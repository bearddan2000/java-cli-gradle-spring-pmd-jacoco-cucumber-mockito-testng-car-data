# java-cli-gradle-spring-pmd-jacoco-cucumber-mockito-testng-car-data

## Description
A POC for a spring app using testng,
jacoco, pmd, cucumber and mockito framework.

## Tech stack
- java
- gradle
	- spring
  - testng
  - cucumber
	- mockito
  - jacoco
  - pmd

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
